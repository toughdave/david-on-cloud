# VLAN & IP Addressing Lab

## Project Summary
Built a hands-on networking lab using enterprise-grade switching equipment and Cisco Packet Tracer to practice VLAN segmentation, structured IP addressing plans, and inter-VLAN routing across isolated network segments.

## Deliverables
- Lab topology diagrams and addressing plans documented per scenario
- Repeatable configuration templates for trunk port, access port, and inter-VLAN routing setups
- Troubleshooting reference covering common VLAN and addressing issues

## Technical Notes
- Scenarios cover trunking configuration, VLAN pruning, spanning tree behavior, and static/dynamic routing integration.
- IP addressing plans use CIDR notation with documented subnet allocation per VLAN segment.
- Physical Cisco switch and Packet Tracer simulations are used together to bridge theory and practice.
